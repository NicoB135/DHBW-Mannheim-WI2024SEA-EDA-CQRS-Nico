# Anleitung für dein neues Smarthome

1. Mit "docker compose up -d" Docker Container für RabbitMQ starten (Docker Desktop sollte offen sein :))
2. Mit ".\mvnw.cmd clean spring-boot:run" im smarthome-Ordner das Programm starten
3. Probier, was in der Kommandozeile steht
4. [bitte anpassen] aktueller Exit: mit exit, aber trotzdem noch Ctrl+C
5. Docker-Container am Ende mit "docker compose down" stoppen

*Viel Spaß im Smarthome!*
